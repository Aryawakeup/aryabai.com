How to Update the Site

You have two options — pick whichever feels easier

Option 1 — Edit directly on GitHub (recommended)
Step 1: Go to github.com → your repo → click the file you want to edit (e.g. index.html)
Step 2: Click the pencil ✏️ icon (top right of the file)
Step 3: Make your changes in the text editor
Step 4: Scroll down → click "Commit changes" → "Commit directly to main"
Result: Your site updates automatically within ~60 seconds ✓

Tip: Press Ctrl+F to find text you want to change — much faster than scrolling
Option 2 — Use this admin panel
This admin panel lets you edit content fields and generates updated HTML you can copy-paste.

Workflow:
1. Edit fields above and click "Save Changes"
2. This generates updated HTML code in the output box below each section
3. Copy that code → go to GitHub → paste into the relevant file → commit

Note: This admin panel itself doesn't need to be uploaded — it runs locally in your browser. Keep it saved on your computer.
Option 3 — Set up Netlify (most powerful)
If you want a proper visual CMS with no code editing ever:

Step 1: Go to netlify.com → "Add new site" → connect your GitHub repo
Step 2: Your site will deploy automatically on every GitHub push
Step 3: Add Netlify CMS or Decap CMS (free) for a visual admin at yoursite.com/admin

This takes ~30 min to set up but then you have a full CMS forever. Ask me to walk you through it when you're ready.
